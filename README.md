<!-- Installation -->

## TERMUX USER
```bash
$ pkg upgrade && pkg update
$ pkg install git -y
$ pkg install nodejs -y
$ pkg install ffmpeg -y
$ pkg install imagemagick -y
$ git clone https://github.com/Rmdhn-20/Gurabot
$ cd Gurabot
$ npm i 
```

```bash
$ npm start
```

#### If npm install failed, try using yarn instead of npm
```bash
$ pkg install yarn -y
$ yarn install
```

## PANEL USER
